 i3 Clipboard History Manager

Программа для управления историей буфера обмена в оконном менеджере i3.

## Описание

Этот инструмент автоматически сохраняет всё, что вы копируете в буфер обмена, и позволяет быстро получить доступ к истории через удобное меню. Программа специально разработана для минималистичного окружения i3wm на Debian.

## Функциональность

- Автоматическое сохранение текста из буфера обмена
- Хранение последних 500 записей
- Защита от дубликатов
- Быстрый поиск по истории через rofi/dmenu
- Очистка всей истории одной командой
- Фоновый режим (демон)
- Уведомления о действиях

## Требования

- Debian/Ubuntu
- Python 3
- xclip
- rofi или dmenu
